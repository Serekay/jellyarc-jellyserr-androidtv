Jellyarc

Android TV Client für Jellyfin mit Jellyseerr Integration

Lade...
Suche neueste Version...

Voraussetzungen

  • Android TV 11 oder neuer
  • Jellyfin Server mit Requests Bridge Plugin

Installation

  • APK auf dem TV herunterladen (z.B. mit BrowseHere Browser)
  • Oder per USB-Stick / FTP auf den TV übertragen
  • "Unbekannte Quellen" erlauben wenn nötig

Features

  • Jellyseerr Discover & Requests direkt in der App
  • Integrierter Tailscale VPN Client
  • Automatische In-App Updates
