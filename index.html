Jellyarc

Android TV Client for Jellyfin with Jellyseerr Integration

What is Jellyarc?

Jellyarc is a Jellyfin client for Android TV with built-in Jellyseerr support. Browse the Jellyseerr discovery catalog and request new content directly from your TV.

Includes an integrated Tailscale VPN client for secure remote access without opening ports.

Requirements

  • Android TV 11 or newer
  • Jellyfin server with Requests Bridge Plugin installed
  • Optional: Tailscale account for remote access

Features

  • Jellyseerr Discover & Request integration
  • Built-in Tailscale VPN client
  • Automatic in-app updates
  • Optimized for TV remote control

Installation

  • Download the APK on your TV using a browser (e.g. BrowseHere)
  • Or transfer via USB stick / FTP
  • Enable "Unknown sources" if prompted
  • Open the APK and install
